# Installation

* Clone the repo
* Create a python [virtual environment](http://docs.python-guide.org/en/latest/dev/virtualenvs/) and activate it (optional, but recommended)
* ```pip install -r requirements.txt```
* Edit _labels.json_ to suit your needs
* ```python manager.py```
