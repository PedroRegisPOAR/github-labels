# Instalação
```shell script
pip install -r requirements.txt
```

# Variáveis de ambiente
Token é um personal token gerado no github https://github.com/settings/tokens
```shell script
TOKEN=random
OWNER=random
REPO=random
```

# Executando
basta executar o comendo
```shell script
python main.py
```
